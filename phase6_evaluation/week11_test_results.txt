Test Objective:
  Verify that the AI Tutor correctly identifies and rewrites all 10
  representative GCC error types from the dataset.

Test Command:
  python phase5_ai_rewriting/rewriter.py errors/<filename>.c

Pass Criteria (all must be true):
  1. GCC error is correctly detected (parser extracts error_dict)
  2. Correct Error ID is matched by the pattern engine
  3. Explanation is returned (non-empty string)
  4. Fix is returned (non-empty string)
  5. secure flag is True
  6. Output is printed in formatted layout

Fail Criteria (any one triggers FAIL):
  - Error not detected (parser failure)
  - Wrong Error ID matched (wrong pattern)
  - Empty explanation or fix returned
  - Program crashes with Python exception
  - secure flag is False
